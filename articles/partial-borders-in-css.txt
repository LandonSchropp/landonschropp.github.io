e:Td9f,<p>Have you ever wanted to create a border in CSS that only extends across part of an element? It's possible by applying margins, but that can get messy when you want your content to extend beyond its border. Here's a handy technique that uses box shadows to achieve the same result.</p>
<p><img src="partial-borders-in-css/example-1.png" alt="Completed example of using box shadows as separators"></p>
<h2>How It's Done</h2>
<p>Typically, when using a border to separate elements in a list, you would do something like this:</p>
<pre><code class="language-scss"><span class="hljs-selector-tag">li</span> {
  <span class="hljs-attribute">border-bottom</span>: <span class="hljs-number">1px</span> solid <span class="hljs-number">#ddd</span>;
}

<span class="hljs-selector-tag">li</span><span class="hljs-selector-pseudo">:last-of-type</span> {
  <span class="hljs-attribute">border-bottom-width</span>: <span class="hljs-number">0</span>;
}
</code></pre>
<p>You can achieve the same effect by using box shadows instead.</p>
<pre><code class="language-scss"><span class="hljs-selector-tag">li</span> {
  <span class="hljs-attribute">box-shadow</span>: inset <span class="hljs-number">0</span> -<span class="hljs-number">1px</span> <span class="hljs-number">0</span> <span class="hljs-number">0</span> <span class="hljs-number">#ddd</span>;
}

<span class="hljs-selector-tag">li</span><span class="hljs-selector-pseudo">:last-of-type</span> {
  <span class="hljs-attribute">box-shadow</span>: none;
}
</code></pre>
<p><img src="partial-borders-in-css/example-2.png" alt="Full-width box shadows"></p>
<p>One nifty aspect of box shadows is that CSS allows you to use more than one of them. You can take advantage of this by stacking two shadows on top of your content.</p>
<pre><code class="language-scss"><span class="hljs-selector-tag">li</span> {
  <span class="hljs-attribute">box-shadow</span>:
    inset <span class="hljs-number">20px</span> <span class="hljs-number">0</span> <span class="hljs-number">0</span> <span class="hljs-number">0</span> <span class="hljs-number">#ee003e</span>,
    inset -<span class="hljs-number">20px</span> <span class="hljs-number">0</span> <span class="hljs-number">0</span> <span class="hljs-number">0</span> <span class="hljs-number">#ee003e</span>,
    inset <span class="hljs-number">0</span> -<span class="hljs-number">1px</span> <span class="hljs-number">0</span> <span class="hljs-number">0</span> <span class="hljs-number">#ddd</span>;
}
</code></pre>
<p><img src="partial-borders-in-css/example-3.png" alt="Box shadows with highlighted stacked borders"></p>
<p>Finally, all you have to do is set the top shadows to be the same color as the background and you're done!</p>
<pre><code class="language-scss"><span class="hljs-selector-tag">li</span> {
  <span class="hljs-attribute">box-shadow</span>:
    inset <span class="hljs-number">20px</span> <span class="hljs-number">0</span> <span class="hljs-number">0</span> <span class="hljs-number">0</span> white,
    inset -<span class="hljs-number">20px</span> <span class="hljs-number">0</span> <span class="hljs-number">0</span> <span class="hljs-number">0</span> white,
    inset <span class="hljs-number">0</span> -<span class="hljs-number">1px</span> <span class="hljs-number">0</span> <span class="hljs-number">0</span> <span class="hljs-number">#ddd</span>;
}
</code></pre>
